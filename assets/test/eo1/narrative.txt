The images were made by placing an SD card into the camera, taking
photos, removing the card, erasing select photos, imaging the card,
and then repeating the process.

Some of the JPEGs are fragmented, some are not. Some are resident in
the file system, some are deleted but recoverable, and several have
data present but no longer have any file system metadata and can only
be recovered through carving. Of these carvable JPEGs at least two are
fragmented.

Each JPEG is a photo of a screen or a piece of paper with prominent
text:
